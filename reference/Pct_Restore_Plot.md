# BMI Percentile Restoration Planning Plot

BMI Percentile Restoration Planning Plot

## Usage

``` r
Pct_Restore_Plot(clean_data, forecast_data)
```

## Arguments

- clean_data:

  a cleaned dataframe - cleaned by the tx_plot_clean function

- forecast_data:

  a forecasted dataframe for 1 participant

- slope_per_week:

  the slope of weight gain expected in lbs per week

## Value

a treatment plan plot
