# Function that converts height from various units (inches, meters, feet) to centimeters (cm).

Function that converts height from various units (inches, meters, feet)
to centimeters (cm).

## Usage

``` r
height_in_cm(height, height_unit = "cm")
```

## Arguments

- height:

  Height value in either inches (in), meters (m), or feet (ft).

- height_unit:

  Unit of height measurement ('in', 'm', or 'ft').

## Value

Height value converted to centimeters (cm).
