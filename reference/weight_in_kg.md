# Function that converts weight from pounds (lb) to kilograms (kg).

Function that converts weight from pounds (lb) to kilograms (kg).

## Usage

``` r
weight_in_kg(weight, weight_units = "kg")
```

## Arguments

- weight:

  Weight value in pounds (lb) or kilograms (kg) (default).

- weight_units:

  Unit of weight measurement ('kg' or 'lbs').

## Value

Weight value converted to kilograms (kg).
