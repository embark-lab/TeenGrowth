# Demo Data

A sample dataset used for demonstrating package functions.

## Usage

``` r
data(demo)
```

## Format

A data frame with the following columns:

- participant:

  Unique identifier for each individual

- age:

  Age in years

- sex:

  Sex (M = male, F = female)

- weight:

  Weight in kilograms

- height:

  Height in centimeters

- adult_height_in:

  Adult height in inches

- ed_aoo:

  Age of onset of eating disorder
